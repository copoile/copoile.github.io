用户状态修改

用户状态修改

请求方法：POST

请求地址：/user/status/update

请求参数：
userId：用户id，必传
status：状态，必传，0:正常，1:锁定，2:禁用，3:过期

需要access_token： 是

需要管理员权限： 是

接口说明：用于禁用、锁定用户等操作。

修改成功：

{
  "code": 0,
  "message": "成功"
}
